5. Groups of Selectors

A comma-separated list of selectors represents the union of all elements selected by each individual
selector in the list. (A comma is U+002C).For example, in CSS, when multiple selectors share the
same declarations, they can be grouped into a comma-separated list. White space may appear before and/or after the comma.

CSS example:

In this example, we condonse three rules with identical declarations into one. Thus,

h1 { font-family: sans-serif; }
h2 { font-family: sans-serif; }
h3 { font-family: sans-serif; }

is equivalent to:

h1, h2, h3 { font-family: sans-serif; }
